เว็บไซต์ SAMA SHOP ของเรามีบริการขายสินค้า บัตร Steam Wallet , KeyGame และ ID GAME ของค่าย STEAM หลังจากลูกค้าได้ทำการ แจ้งชำระเงิน และรอสถานะสินค้าแจ้งว่า "สำเร็จแล้ว" รอรับสินค้าที่หน้า รายการสั่งซื้อ ของลูกค้าเองได้ครับ (หากเกิดปัญหาหรือไม่เจอสามารถทักแชท messenger เพจของทางเว็บไซต์ได้เลยครับ ) มี 3 สถานะ = 1) รอการตรวจสอบ , 2) ตรวจสอบสำเร็จ และ 3) ตรวจสอบไม่สำเร็จ (โดยจะมีแจ้งเหตุผลด้วย) สินค้าประเภทที่รับประกัน
ID GAME (ระยะเวลา 14 วัน)


สินค้าประเภทที่ไม่รับประกัน
Steam Wallet , Key Game และ โปรแกรม หากเกิดปัญหาลูกค้าสามารถแจ้งรายละเอียดมาได้ที่ แชท messenger เพจของทางเว็บไซต์ และรอทางแอดมินตรวจสอบ คาดว่าใช้เวลาการตรวจสอบไม่เกิน 3-7 วัน (ไม่มีวันหยุด)
